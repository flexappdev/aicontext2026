# AI Apps 2026

The essential AI applications in daily use across coding, research, content, and media.

---

## Claude (Anthropic)

- **claude.ai** — Chat, Projects, Artifacts, voice mode, file uploads
- **Claude Code** — CLI agentic coding (`npm i -g @anthropic-ai/claude-code`)
- **API** — `claude-opus-4-6`, `claude-sonnet-4-6`, tool use, vision, 200K context

| Feature | Detail |
|---------|--------|
| Projects | Persistent context across conversations |
| Artifacts | Code, SVG, HTML previewed inline |
| Extended thinking | Opus 4.6 shows reasoning chain |
| Voice | Real-time voice conversation |

---

## ChatGPT (OpenAI)

- **chatgpt.com** — GPT-5, o3, DALL-E 4, voice, canvas, file analysis
- **Operator** — AI agent with web browsing + code execution
- **Sora** — Video generation from text or image

| Feature | Detail |
|---------|--------|
| Canvas | Collaborative document/code editor with AI |
| Memory | Remembers preferences across sessions |
| Custom GPTs | Specialized assistants via GPT store |
| Plugins | Web browsing, code interpreter, DALL-E |

---

## Gemini (Google)

- **gemini.google.com** — Gemini 2.5 Pro, Deep Research, canvas, Workspace integration
- **AI Studio** — aistudio.google.com — prototyping, API testing, Gemini free tier
- **NotebookLM** — notebooklm.google.com — sources → podcast, slides, infographic, video
- **Veo 3** — State-of-art video generation with audio
- **ImageFX** — Imagen 3 image generation

| Feature | Detail |
|---------|--------|
| Deep Research | Multi-step research with citations |
| 1M context | Entire codebases, books in one context |
| Gems | Custom Gemini personas |
| Google integration | Search, Docs, Gmail, YouTube grounding |

---

## Grok (xAI)

- **grok.com** — Grok 3, real-time X/web data, Aurora image gen, voice
- No knowledge cutoff — live internet access
- Strong at: current events, humor, uncensored responses

---

## Perplexity

- **perplexity.ai** — AI search with cited sources, real-time web
- **Deep Research** — multi-step research reports
- API available for developers

---

## Antigravity

- VS Code extension for AI-accelerated app development
- Skills, agents, context management inside editor
- Works alongside Claude Code and Cursor

---

## Cursor / Windsurf

- **Cursor** — AI-native VS Code fork, Composer for multi-file edits
- **Windsurf** — Codeium's IDE, Cascade agentic mode
- Both support Claude, GPT-4o, Gemini via model selector
