# AI Apps 2026

## Claude (Anthropic)
- **claude.ai** — Chat, Projects, Artifacts, voice
- **Claude Code** — CLI agentic coding (`npm i -g @anthropic-ai/claude-code`)
- **API** — `claude-sonnet-4-6`, tool use, vision, streaming

## ChatGPT (OpenAI)
- **chatgpt.com** — GPT-5, o3, image gen (DALL-E 4), voice
- **Operator** — AI agents with web browsing, code execution
- **Sora** — Video generation from text/image

## Gemini (Google)
- **gemini.google.com** — Gemini 2.5 Pro, Deep Research, canvas
- **AI Studio** — gemini.google.com/app — prototyping, API testing, free tier
- **NotebookLM** — notebooklm.google.com — sources → podcast, slides, infographic

## Antigravity
- VS Code extension for AI-accelerated app development
- Skills, agents, context management inside editor
- Link: vscode extension / antigravity.dev

## Grok (xAI)
- **grok.com** — Real-time X/web data, image generation (Aurora)
- Strong at current events, no knowledge cutoff

## Perplexity
- AI search with citations
- Real-time web answers

## Cursor / Windsurf
- AI-native code editors (fork of VS Code)
- Composer: multi-file edits with context

## Earth Studio (Google)
- **earth.google.com/studio** — Cinematic 3D earth animations
- Export for video backgrounds and intros
